# 📊 Filtro e Análise — StoreOrder

Este diretório contém **exclusivamente** o código utilizado para filtrar, analisar e refinar os dados da planilha **StoreOrder**, com o objetivo de gerar o documento final localizado em:

```
documento → Entrega_01 → Contabilidade_E_Financas
```

---

## 🛠️ Tecnologias utilizadas

- **Python**
- **FastAPI** — framework web para construção da API
- **Pandas** — manipulação e análise dos dados da planilha

---

## ▶️ Como testar

1. Navegue até este diretório no terminal:

```bash
cd caminho/para/este/diretorio
```

2. Instale as dependências necessárias:

```bash
pip install "fastapi[standard]"
```

3. Inicie o servidor em modo de desenvolvimento:

```bash
fastapi dev
```

---

> **Observação:** Este código tem finalidade exclusivamente de processamento de dados intermediário. O resultado gerado alimenta o documento final de **Contabilidade e Finanças** da Entrega 01.
